    public silence() source

    Silence this entity, so ValueTracker based events are no longer broadcast from this entity. Once this is done on an entity, it cannot be undone. It should be done before discarding an entity for good.

    Example:

        assert( !lyph.deleted );
    lyph.delete();
    assert(  lyph.deleted );
